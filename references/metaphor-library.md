# Metaphor Library

Use this as a starting point, not a mandatory lookup table. Match the mechanism, not only the topic.

| Abstract mechanism | Concrete visual families |
|---|---|
| Capacity limit | desk, shelf, container, suitcase, parking lot |
| Chunking or tokenization | building blocks, slices, parcels, conveyor belt |
| Retrieval | library, archive drawers, searchlight, map pins |
| Similarity | objects clustering by shape/color, magnetic attraction |
| Context loss | older objects pushed out, fading notes, overflowing desk |
| Hallucination | confidently completing a puzzle with the wrong piece |
| Agent/tool use | assistant selecting and operating tools at a workbench |
| Workflow | stations, kitchen recipe, assembly line |
| Compression | packing a suitcase, folding a map, making a summary card |
| Branching choice | road junction, switchboard, decision gates |
| Feedback loop | thermostat, circular conveyor, iterative sketch |
| Input/output | mailbox and delivery, question/answer pipes |

## Scene Formula

Build scenes with:

```text
subject + action + object + visible consequence
```

Weak:

```text
A robot with blocks.
```

Strong:

```text
A small robot separates one long strip into four building blocks, then sends the blocks through a compact processing machine.
```

Prefer a single readable action. Avoid a collage of unrelated symbols.
